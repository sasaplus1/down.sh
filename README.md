# down.sh

jump to sub dir

## Installation

append to your `.bashrc` or `.bash_profile` the following:

```sh
. /path/to/down.sh
```

## Usage

```sh
$ down path
```

if you want to use [peco](https://github.com/peco/peco):

```sh
$ down -p path
```
